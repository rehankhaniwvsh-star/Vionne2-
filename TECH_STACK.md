# Tech Stack - VIONNE

## 1. Frontend
- **Framework**: React 19
- **Build Tool**: Vite 6
- **Language**: TypeScript 5.8
- **Styling**:
  - Tailwind CSS 4 (Utility-first framework)
  - Framer Motion (Animations and transitions)
  - Lucide React (Icon library)
- **State Management**: Zustand 5 (Lightweight state management)
- **Routing**: React Router DOM 7 (Client-side navigation)
- **Data Visualization**: Recharts 3 (Charts and analytics)

## 2. Backend
- **Runtime**: Node.js
- **Framework**: Express 4 (Web server)
- **Language Support**: tsx (TypeScript execution for Node.js)
- **Environment Management**: dotenv (Environment variables)

## 3. Database & Services
- **Database**: Firebase Firestore (NoSQL document database)
- **Authentication**: Firebase Authentication (Google Login)
- **Admin SDK**: Firebase Admin (Server-side Firestore access)
- **AI Integration**: @google/genai (Gemini AI for product descriptions or customer support)

## 4. Development & Tooling
- **Linting**: TypeScript compiler (`tsc --noEmit`)
- **Package Manager**: npm
- **Version Control**: Git
- **Deployment**: Google Cloud Run (Containerized hosting)

## 5. Key Dependencies
- `clsx` & `tailwind-merge`: Utility for conditional Tailwind classes.
- `firebase`: Client-side Firebase SDK.
- `firebase-admin`: Server-side Firebase SDK.
- `motion`: Framer Motion for React 19.
- `react-router-dom`: Routing for SPAs.
- `zustand`: State management for cart and UI.
- `recharts`: Charting library for admin analytics.
